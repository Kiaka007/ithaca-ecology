# Academy Memory Index

This directory holds sphere-local memory for the Academy. Claude reads these files on arrival and updates them on departure.

Files to create and maintain:

- `threads-index.md` — Active threads: name, status, last-touched date, one-line summary
- `commissions-state.md` — Active commissions: status, next action, blockers
- `patterns.md` — Recurring intellectual themes across threads and experiments
- `promotion-watch.md` — Ideas that are moving toward a different mode
- `blockers.md` — What is stuck and why
- `[advisor-a]-sessions.md` — Ideas opened by your generative faculty advisor
- `[advisor-b]-sessions.md` — Claims tested by your critical faculty advisor
