# Address Book

Two purpose-built structures, kept separate:

- **`sources/`** — trusted institutions, journals, publications, and
  thought-leaders, one file per subject domain. First-stop references for
  web lookups in a matching subject area.
- **`people.yaml`** — individuals with consistent contact (friend, colleague,
  collaborator, family). Identity resolution and relationship-standing, not
  a detailed contact log.

See `CLAUDE.md` in this directory for field definitions and usage
instructions. Templates: `bag/tools/templates/source-contact.yaml` and
`bag/tools/templates/person-contact.yaml`.
