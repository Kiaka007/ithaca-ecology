# Woodlands Memory Index

This directory holds sphere-local memory for the Woodlands. Claude reads these files on arrival and updates them on departure.

Files to create and maintain:

- `seasonal-observations.md` — What was noticed or practiced near each sabbat; brief seasonal record
- `element-log.md` — Running log of which elements (earth, wind, water, fire) have been active
- `signals.md` — Recurring wellbeing patterns worth holding across time
